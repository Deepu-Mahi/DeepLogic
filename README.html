🚀 DeepLogic

DeepLogic is a powerful Online Judge platform and code compiler system. It provides a space for users to solve coding problems, run code in multiple programming languages, and get real-time feedback. Built with Django backend and a responsive Bootstrap frontend.

🌟 Features

  • 🔐 User registration, login, and profile management
  • 📚 Problem list with detailed descriptions and constraints
  • ✍️ Code editor with support for Python, C, C++, and Java
  • 🧪 Automatic code execution and test case validation
  • 📜 Submission history tracking per user
  • 🧠 AI support for debugging and explanation (OpenAI/Gemini)
  • ⚙️ Admin panel to manage problems and test cases
  • 💡 Responsive and clean UI using Bootstrap

🧰 Tech Stack

LayerTechnology
BackendDjango (Python)
FrontendHTML, CSS, JavaScript, Bootstrap 5
DatabaseSQLite / PostgreSQL

DeploymentRailway / Vercel / Docker

🚀 Getting Started

1. Clone the Repository
git clone https://github.com/Deepu-Mahi/DeepLogic.git
cd DeepLogic
2. Setup Virtual Environment
python -m venv env
source env/bin/activate   # Linux/Mac
env\Scripts\activate      # Windows
3. Install Requirements
pip install -r requirements.txt
4. Migrate Database
python manage.py migrate
5. Create Superuser
python manage.py createsuperuser
6. Run the Server
python manage.py runserver

Open your browser at: http://127.0.0.1:8000/
